# Projeto de Aplicação de Java e POO :coffee: :computer:

## Sobre :memo:

Este projeto foi desenvolvido para aplicar os conhecimentos adquiridos em **Java** e para demonstrar a aplicação de **Programação Orientada a Objetos (POO)** no desenvolvimento.

## Objetivos :dart:

- Aplicar e demonstrar os conceitos de **Java** no desenvolvimento de software.
- Utilizar os princípios da **Programação Orientada a Objetos (POO)** para criar um código estruturado e reutilizável.
- Desenvolver habilidades práticas e teóricas em Java e POO.

## Tecnologias Utilizadas :computer:

- **Java**: A linguagem de programação utilizada para desenvolver o projeto.
- **POO**: Os conceitos de Programação Orientada a Objetos foram aplicados para estruturar o código.

## Como Executar :play_or_pause_button:

1. Clone o repositório para a sua máquina local.
2. Navegue até o diretório do projeto.
3. Execute o comando `java NomeDoArquivo.java` para compilar e executar o projeto.
